# 面经
* 简历上的东西很重要，面试前需要准备好在简历上写的东西
* 应该把面试题背熟，回答时快速流畅，不应该有停顿的语气词，例如“呃”，“这个...”，“那个...”，“然后”，“啊”之类的
* 自信，侃侃而谈，当然这需要你的知识要足够丰富
* 应学习比较深的知识，例如react内核原理，业务知识，webpack打包知识
* 最好的情况就是每个问题都能答上来，按照是什么，作用，好处/坏处来说明
* 回答不能少也不能多，把该答的回答上来，就能给分，答不到点上，照样扣分，千万不要歪曲面试官的意图，在面试官看来你就是不会
* 需要对知识有很深的认识，在回答时要说比较深一点，超出面试官的预期
* 需要说应用，自己是怎么用的
