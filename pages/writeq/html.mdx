## 如何使用一个监听给ul下面多个li绑定监听事件
当逐个给每个li绑定监听事件，当li太多之后，需要绑定很多监听器，如果有1000个li，因此需要事件代理，又称事件委托。
简单的来讲就是利用JS中事件的冒泡属性，把原本需要绑定的事件委托给父元素，让父元素担当事件监听的职务。
```js
let ul = document.getElementById('ul')
ul.addEventListener('click', (e) => {
    let t = e.target
    switch(t.innerText) {
        case '1':
            break
        case '2':
            break
    }
})
```
